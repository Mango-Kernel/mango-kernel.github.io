  • Getting Started

    Install the following software to build:

    • GCC
    • Git
    • QEMU (optional)
    • Make
    ArchLinux

    If you haven't installed yay, follow this guide

    pacman -S make git
yay -S i686-elf-gcc

    To install QEMU, run:

    pacman -S qemu

    To clone the repository, run:

    git clone https://github.com/winvistalover/PotatoOS.git
  • Building

    To build the operating system, run:

    sh build.sh

    To build the operating system and make an iso, run:

    sh iso.sh
  • Running

    To run the operating system, run:

    sh qemu.sh

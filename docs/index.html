  • Welcome to the PotatoOS documentation (work in progress)

    PotatoOS uses it's own lightweight multiboot kernel
    The source code for PotatoOS can be found here.
    • Getting Started

      Install the following software to build:

      • GCC
      • Git
      • QEMU (optional)
      • Make
      • Arch Linux

        If you haven't installed yay, follow this guide

        pacman -S make git
yay -S i686-elf-gcc

        To install QEMU, run:

        pacman -S qemu

      To clone the repository, run:

      git clone https://github.com/winvistalover/PotatoOS.git
    • Building

      To build the operating system, run:

      sh build.sh

      To build the operating system and make an ISO, run:

      sh iso.sh

      This will output "mango.elf" in the root directory of the ISO image
    • Running

      To run the operating system, run:

      sh qemu.sh

      Or:

      qemu-system-i386 -cdrom potatoOS.iso
    • Modifying

      The operating system contains multiple files (tty.c, vga.h, etc) in the "kernel/arch/i386" directory.

      To modify the operating system, you can edit these files using a text editor (e.g, nano).

      Each file serves a different purpose:

      File Description
      kernel/arch/i386/tty.c Contains the code for the terminal
      kernel/arch/i386/vga.h Contains the code for the VGA driver
      kernel/arch/i386/boot.S Contains the code for a bootloader (e.g, GRUB) to load
      kernel/arch/i386/linker.ld Contains the code for the linker
